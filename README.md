# Tint

Tint is an EVM-focused utxo-based proof-of-concept privacy protocol built with circom. It's designed with the following intentions:
- [x] Hyper-optimized commitment insertion costs, with a target of 80k gas.
- [x] Flexible output options, enabling multiple unshields per operation.
- [ ] Flexible note ownership, enabling single-ownership, multisigs, timelocks, p2p swaps, and more.
- [ ] 4337 paymaster support, enabling gasless transactions and paying fees with private assets.
